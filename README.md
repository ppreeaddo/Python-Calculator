# Python Calculator

This is a simple calculator I built in Python that performs addition, subtraction, multiplication, and division. It includes input handling and error checking to prevent division by zero.

## Features
- Addition, subtraction, multiplication, division
- Handles decimal numbers
- Prevents division by zero
- User-friendly input prompts

## How to Run
1. Make sure Python 3.11(recommended) is installed on your computer.  
2. Open the project in an IDE of your choice (VS Code, PyCharm, or any other Python IDE).  
3. Open the Python file (`Python Calculator Project.py`) in the IDE.  
4. Run the file to start the calculator. Follow the on-screen prompts to enter numbers and choose operations:
